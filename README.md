# E-Commerce-Back-End
This app serves as the back-end for an E-Commerce Website. Express.js was used for the server side and MYSQL for the database. Sequelize was used for the ORM to run SQL models.

# Technologies used
- Express.js
- Node.js
- MYSQL
- Sequelize
- Dotenv
